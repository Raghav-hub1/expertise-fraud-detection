expertise-fraud-detection

This repository contains applied statistics projects implemented using Python.
The goal is to demonstrate statistical reasoning on real and simulated data,
with clear explanations and reproducible analysis.

## Topics Covered
- Probability simulations
- Exploratory Data Analysis (EDA)
- Hypothesis testing
- Regression analysis
- Statistical interpretation

## Tools Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- SciPy
- Statsmodels

## Background
This repository is developed by a BSDS student at the Indian Statistical Institute (ISI Bangalore)
as part of internship preparation and applied learning.

## Note
Focus is placed on interpretation and reasoning rather than only model accuracy.
